# Быстрый старт

## Установка и запуск
```bash
npm install
npm run dev
```

## Сборка для production
```bash
npm run build
# Файлы в папке dist/ готовы к развертыванию
```

## Ключевые файлы
- `src/lib/i18n.tsx` - тексты на 3 языках
- `src/pages/` - страницы сайта
- `public/images/` - изображения и логотипы
